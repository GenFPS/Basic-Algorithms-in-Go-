# Префиксные суммы

Префиксными суммами массива [a_0, a_1, a_2, ..., a_n-1] называется массив [s_0, s_1, s_2, ..., s_n-1], в котором:

- s_0 == a_0
- s_1 == a_0 + a_1
- s_2 == a_0 + a_1 + a_2
- ...
- s_n-1 == a_0 + a_1 + a_2 + ... + a_n-1

## Задача.

Условие: Дан массив целых чисел. Найти сумму на полуинтервале с позиции l до r за O(1).
